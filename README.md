## ⚡ Getting Started  

```bash
# Clone the repository
git clone https://github.com/TheAlgorithms/Python.git

# Navigate into the folder
cd Python

# Install dependencies
pip install -r requirements.txt

# Run an algorithm
python sorts/bubble_sort.py
